EMMET : Un ami qui vous veut du bien

EMMET permet à l'aide d'une seule ligne de générer du code HTML, par exemple : h1+h2+p va nous donner un premier titre principal, puis un deuxième titre puis un paragraphe !

Autre exemple : lorsque je veux rajouter une classe à une balise div je n'ai qu'à faire div.nom-de-ma-classe. Et bim une div est créee avec le nom de classe desiré !

Également, si je fais div#ma-div>(h1+h2+p*3) cela va me donner une div avec l'id "ma-div" contenant un h1, un h2 et 3 paragraphes.

Tips supplémentaire : Pour ajouter du contenu entre les balises d'un élément je peux le mettre entre crochet.
ul>li*2{test} me donnera une ul avec dedans deux li qui auront pour contenu "test". Si je remplace {test} dans l'exemple précédent par {test $} cela m'ajoutera un décompte après le mot test : test 1 puis test 2
